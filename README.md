Game Level - Gorilla in the Jungle

As of currently, this project is unfinished, although the program can be run in a stable condition.

The gorilla character has movement and sprite animations for WASD inputs,
along with dialogue when the banana or tree objects are interacted with 
using space bar.

The ESC key can be used to exit the program.
